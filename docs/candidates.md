# 国产支持升降压且支持 USB PD 的电源控制芯片清单（结构化）

用途：为 USB‑PD 源端/双向快充（移动电源、便携电源、桌面电源等）做初选与比对。所有候选条目均为“平级”，按同一标准入围并统一对比。量产立项前请以原厂数据手册/代理反馈为准。

## 入围标准

- 国产厂商器件，且“具备升降压能力”：
  - 集成 SoC（含升降压+PD 协议），或
  - 通过“升降压控制器 + PD 协议 IC”组合实现双向路径。
- 支持 USB PD（≥PD3.0；若支持 PD3.1/EPR/AVS 将单独标注）。
- 公开资料可核对（数据手册、权威媒体拆解或代理技术页）。
- 给出适配的电池串数与目标功率段，适合源端快充应用。
- 量产可行性：优先当前在产/可供；如为 EOL 明确标注并仅作参考。
- 对“需协议 IC 配合”的候选，需满足下列其一：
  - 支持 FB 方案调节，或
  - 在候选条目中明确列出可配合的协议 IC 型号（已量产/已验证），并注明调压方式（FB/I2C/其他）。

### 智融 SW7203（升降压控制器，需协议 IC）

- 介绍：智融首批面向 1–4 串锂电的双向升降压控制器，定位 100W 快充平台，支持 NVDC 路径管理与外部协议搭配，实现双向充放电管理（来源：PPMY 智融方案稿）。
- 特性：四管拓扑、最高 97% 效率、集成适配器/负载插入检测、12 位 ADC、I2C 接口，可与 MCU/协议 IC 组成完整方案（来源同上）。
- 适用场景：100W 等级车载快充、移动电源、户外储能等需要 1–4 串升降压管理的产品（来源同上）。
- 性能指标：支持 1–4 串锂电池，充放电功率覆盖至 100W，单芯片双向升降压输出（来源同上）。
- 调压方式：I2C 配置 + NVDC 路径管理，支持外部协议侧 AVS/电压调节（来源同上）。
- 可配协议 IC：智融 SW2303 等协议芯片组合车充/移动电源解决方案（来源同上）。

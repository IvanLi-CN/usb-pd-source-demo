# General Description

# Features

The AON7410 uses advanced trench technology and design to provide excellent ${ \sf R } _ { \sf D S ( O N ) }$ with low gate charge. This device is suitable for use in DC - DC converters and Load Switch applications.

$\mathsf { V } _ { \mathsf { D } \mathbb { S } } \left( \mathsf { V } \right) = 3 0 \mathsf { V }$ I = 24A $( \mathsf { V } _ { \mathsf { G S } } = 1 0 \mathsf { V } )$ $\mathsf { R } _ { \mathsf { D S ( O N ) } } < 2 0 \mathsf { m } \Omega$ $( \mathsf { V } _ { \mathsf { G S } } = 1 0 \mathsf { V } )$ $R _ { \tt D S ( O N ) } < 2 6 \mathrm { m } \Omega$ $( \mathsf { V } _ { \mathsf { G S } } = 4 . 5 \mathsf { V } )$

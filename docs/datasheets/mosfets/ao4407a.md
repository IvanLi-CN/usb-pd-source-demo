# General Description

# Product Summary

The AO4407A uses advanced trench technology to provide excellent $\mathsf { R } _ { \mathsf { D S } ( \mathsf { O N } ) }$ , and ultra-low low gate charge with a 25V gate rating. This device is suitable for use as a load switch or in PWM applications.

$\mathsf { V } _ { \mathsf { D S } } = - 3 0 \mathsf { V }$ $\mathsf { I } _ { \mathsf { D } } = - 1 2 \mathsf { A }$ $( \mathsf { V } _ { \mathsf { G S } } = - 2 0 \mathsf { V } )$ $\mathsf { R } _ { \mathsf { D S ( O N ) } } < 1 1 \mathsf { m } \Omega$ $( \mathsf { V } _ { \mathsf { G S } } = - 2 0 \mathsf { V } )$ $\mathsf { R } _ { \mathsf { D S ( O N ) } } < 1 3 \mathsf { m } \Omega$ $( \mathsf { V } _ { \mathsf { G S } } = - 1 0 \mathsf { V } )$ $R _ { \sf D S ( O N ) } < 1 7 \mathsf { m } \Omega$ $( \mathsf { V } _ { \mathsf { G S } } = - 6 \mathsf { V } )$ ）

# Description

The NCE40P13S uses advanced trench technology and design to provide excellent ${ \mathsf { R } } _ { { \mathsf { D } } { \mathsf { S } } ( { \mathsf { O N } } ) }$ with low gate charge. It can be used in a wide variety of applications.

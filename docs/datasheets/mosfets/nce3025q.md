# NCE N-Channel Enhancement Mode Power MOSFET

# Description

The NCE3025Q uses advanced trench technology and design to provide excellent ${ \mathsf { R } } _ { { \mathsf { D } } { \mathsf { S } } ( { \mathsf { O N } } ) }$ with low gate charge. It can be used in a wide variety of applications.

# General Features

$\mathsf { V } _ { \mathsf { D } \mathsf { S } } = 3 0 \mathsf { V } , \mathsf { I } _ { \mathsf { D } } = 2 5 \mathsf { A }$ $\mathsf { R } _ { \mathsf { D S } ( \mathrm { O N } ) } < 1 0 \mathsf { m } \Omega \ @ \mathsf { V } _ { \mathsf { G S } } = 1 0 \mathsf { V }$ $R _ { \mathsf { D S ( O N ) } } < 1 4 \mathsf { m } \Omega \ @ \mathsf { V } _ { \mathsf { G S } } = 4 . 5 \mathsf { V }$ $\bullet$ High density cell design for ultra low Rdson $\bullet$ Fully characterized avalanche voltage and current $\bullet$ Good stability and uniformity with high $\mathsf { E } _ { \mathsf { A S } }$ $\bullet$ Excellent package for good heat dissipation $\bullet$ Special process technology for high ESD capability

# Application

$\bullet$ SMPS and general purpose applications $\bullet$ Hard switched and high frequency circuits $\bullet$ Uninterruptible power supply

# Description

The NCEP6080AG uses Super Trench technology that is uniquely optimized to provide the most efficient high frequency switching performance. Both conduction and switching power losses are minimized due to an extremely low combination of ${ \mathsf { R } } _ { { \mathsf { D } } { \mathsf { S } } ( { \mathsf { O N } } ) }$ and $\mathsf Q _ { \mathsf g }$ .This device is ideal for high-frequency switching and synchronous rectification.
